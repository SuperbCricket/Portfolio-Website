Introduction

A Platformer seems like a pretty easy game to buid for most people, which might be true but making a platform feel good takes a lot of practice. Most people think a platform just exists out of Jumping and walking, but this is not always the case.

What is this project about

This Project is no meant to become a game at all but mostly to test and theorize movement so that I'll be able to use this knowledge to build good movement in the future.

State Machine

The first step to making a good platformer is making a state machine that can tell other movement related scripts in what the state will be. this is very important. Because without a proper state machine a players might be able to do things you don't want them to. for example: Jumping while being airborn.
